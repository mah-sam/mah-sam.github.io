Led development of an end-to-end pipeline that converts hand-drawn circuit diagrams into simulatable netlists. The system chains fine-tuned YOLOv11 for component detection, custom OpenCV algorithms for connectivity analysis, SAM 2 for segmentation, and the Gemini API for component value recognition. Awarded **2nd Best Graduation Project in AI** university-wide at Islamic University of Madinah.
