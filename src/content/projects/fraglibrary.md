Built and deployed a full-stack fragrance discovery platform covering 10,000+ fragrances. Trained a custom embedding model (Scent2Vec 2.0) that encodes scent descriptors into a vector space, enabling cosine-similarity-based preference matching. Deployed on Vercel with a Django REST backend and PostgreSQL database.
